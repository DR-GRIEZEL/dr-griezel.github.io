# dr-griezel.github.io

CRT-styled personal dashboard website with a retro terminal aesthetic.

## Website features
- CRT-inspired UI (high-contrast borders, glow effects, scanlines).
- Responsive sidebar navigation.
- Dashboard widgets for analog clock, weather, and pomodoro.
- Updates feed pulled from GitHub.
- Lightweight static pages for status and login placeholders.

## Development
- Static HTML/CSS/JS with Jekyll layouts.
- Tests run with Vitest.

## Tests
```sh
npm test
```
